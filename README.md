# AV1Studio

A GUI for AV1 encoding via Av1an and SVT-AV1-PSY written in Rust using iced.

## Building

To build **AV1Studio**, you only have to do this:

```bash
git clone
cd AV1Studio/
cargo build --release
```

The built binary will be located at `./target/release/av1studio`.
